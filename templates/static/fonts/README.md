# Fuentes

Descargá Inter desde https://fonts.google.com/specimen/Inter y colocá
`Inter-Regular.ttf` e `Inter-Bold.ttf` en este directorio.

Alternativa: la plantilla usa sans-serif del sistema si faltan estos
archivos, pero la tipografía puede verse distinta según el entorno donde
corra WeasyPrint (WSL2, servidor Linux, Mac, etc.).

## Archivos esperados

- `Inter-Regular.ttf` (weight 400)
- `Inter-Bold.ttf` (weight 700)

Son los dos únicos weights que usa `templates/reporte_poligono.html`.
